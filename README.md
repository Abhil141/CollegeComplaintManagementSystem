# College Complaint Management System

## Overview

This project is a desktop-based complaint management system developed using Java Swing, JDBC, and SQL Server. The system allows students, faculty, and staff to submit complaints, track complaint status, and manage issue resolution through an administrative interface.

The system was designed using object-oriented programming principles and a database-backed architecture to manage complaints efficiently.

---

## Features

* User login system
* Submit complaints
* Track complaint status
* Admin complaint management
* Email notifications for complaint updates
* Database storage using SQL Server
* Desktop GUI using Java Swing

---

## System Architecture

Java Swing GUI → Java Backend → JDBC → SQL Server Database → Email Notification System
